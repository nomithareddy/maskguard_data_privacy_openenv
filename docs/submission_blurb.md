# Submission Blurb

MaskGuard Data Privacy OpenEnv is an OpenEnv-compatible environment for dataset privacy and readiness remediation. It simulates real data-governance issues such as exposed PII, missing values, duplicate rows, schema problems, and bias signals, then evaluates whether an agent can take the right remediation actions to improve dataset readiness.

The project includes a FastAPI-based OpenEnv server, structured action and observation models, multi-difficulty task scenarios, local grading, and evaluator-style inference logs. It is designed as a compact benchmark for privacy-aware data cleanup and multi-step remediation workflows.
