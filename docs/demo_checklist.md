# Demo Checklist

## Recommended live demo sequence

- Show the repository and README
- Run `python3 pre_validation.py`
- Run `python3 inference.py`
- Run `python3 baseline/run_baseline.py`
- Explain the observation space and supported actions
- Explain why this is useful for privacy-aware agent evaluation

## Optional talking points

- Multi-step remediation instead of one-shot scoring
- Support for privacy, quality, and bias signals in one environment
- OpenEnv-compatible FastAPI server for deployment and experimentation
